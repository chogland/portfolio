Introducing the Intel® Core™ i+ Processor Family

Brace yourself for something special: The Intel® Core™ i+ processor family is coming, which means customers are going to have an exciting new way to enjoy premium performance that’s optimized to work the way they do!1

Overview

Intel® Core™ i+ processors combine the performance of select 8th Gen Intel® Core™ i5, i7, and i9 processors with the responsiveness of Intel® Optane™ technology!

8th Gen Intel® Core™ processors deliver uncompromising performance. And now, with Intel® Optane™ technology built in, 8th Gen Intel® Core™ i+ processors can deliver an amazingly responsive experience that’s optimized based on how customers use their PCs. 1

Note that customers can still purchase separate, boxed Intel® Optane™ SSDs or memory modules to add to select desktops.

With Intel® Core™ i+ processors, they’ll be able to get all the advantages of Intel® Optane™ technology integrated into a variety of the latest desktops and notebooks!

Intel® Optane™ Technology

As a reminder, there are currently two types of Intel® Optane™ technology, which deliver different benefits. The Intel® Core™ i5+/i7+/i9+ platform combines either Intel® Optane™ memory or Intel® Optane™ SSDs with select 8th Gen Intel® Core™ processors.

Memory

Intel® Optane™ memory is a radically innovative technology that makes computers work more efficiently. It’s a smart, adaptable system accelerator that—when paired with a SATA-based storage device—provides greater performance while simultaneously personalizing the programs and commands customers use most, so they can launch faster, play smoother, and work with ease.2,3

Storage

Intel® Optane™ SSDs are designed for the most storage-demanding workloads, delivering exceptionally high random read/write performance and low latency for professional users, creators, and enthusiasts!1

Details May Appear on Fact Tags

Some fact tags will show whether a system has Intel® Optane™ memory or, less commonly, Intel® Optane™ SSDs. Note that:

  • If present, Intel® Optane™ memory will be listed on the memory line.
  • The “Total Memory” icon will show the total of Intel® Optane™ memory plus DRAM.

If the fact tag doesn’t show total system memory in this way, you can still speak to the benefits that Intel® Optane™ technology provides!

Customer Types

Let’s look at how Intel® Core™ i+ processors with Intel® Optane™ memory can benefit customers who are interested in gaming, content creation, and productivity.

Productivity-minded customers

Intel® Core™ i+ processors with Intel® Optane™ memory will deliver the performance and responsiveness to be productive, including the ability to: 1

  • Boot up, launch applications, and perform compute-intensive tasks quickly, smoothly, and easily
  • Accelerate the most frequently performed tasks
  • Multitask with ease, even when working with large files and demanding applications

Content creators

Intel® Core™ i+ processors with Intel® Optane™ memory will enable content creators to: 1

  • Enjoy automatic acceleration of their most frequently used content creation applications
  • Launch professional-grade content creation apps at SSD-like speeds—and still have lots of storage capacity
  • Rapidly create, edit, share, store, and access even huge digital libraries

Gamers

With Intel® Core™ i+ processors with Intel® Optane™ memory, gamers will be able to: 1

  • Install: All games can be installed on a single, large-capacity hard drive and play at SSD-like speeds.
  • Launch: Automatic acceleration speeds the launch of customers’ most frequently played games.
  • Level up: Gamers can load levels at incredible speeds!
  • Play: Even at high settings, customers can enjoy smooth gameplay.
